# BYTE بیٹھک Website

A premium, modern event website for BYTE بیٹھک - bridging cutting-edge fintech innovation and emerging tech talent at UET Lahore.

## Design Philosophy

- Apple-level minimalism meets Google Dev professionalism
- Clean spacing, strong hierarchy
- Tech-forward, modern, confident
- Zero fluff, zero generic templates
- Dark theme with sophisticated plum/wine color palette

## Tech Stack

- React 18.3.1
- Vite 5.4.11 (for blazing fast development)
- Pure CSS (no framework dependencies)
- Custom design system with CSS variables

## Color Palette

- **Plum Dusk** (#5A1761) - Primary brand color
- **Deep Wine** (#33193F) - Dark variant
- **Burgundy Noir** (#2A0E2B) - Rich depth
- **Blackberry** (#1F0A1C) - Deepest shade
- **Midnight** (#0B0B0B) - Pure dark background

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/          # All section components
│   ├── Hero.jsx         # Landing with countdown timer

│   ├── WhoShouldAttend.jsx     # Target audience

│   └── FinalCTA.jsx     # Registration call-to-action

```
